# Commands
## Install dependencies
```
npm install
```

## Run the project
```
npm run serve
```

## Launch unit tests
```
npm run test
```

## Launch coverage
```
npm run coverage
```

Then open up the file : `coverage/lcov-report/index.html`. Note: `coverage` will be generated at the root of the project.
